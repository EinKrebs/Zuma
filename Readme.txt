Задача "ZUMA"
Автор - Фролов Юрий ФТ-101

Управление:
A - поворот турели влево
D - поворот турели вправо
Space - выстрел
Mouse Whell Up - поворот турели влево
Mouse Wheel Down - поворот турели вправо
Mouse Left - выстрел

Уровни задаются в файле level_description.txt.
Каждый уровень задаётся в следующем формате:
    -сначала задаюися параметры эллипса: ширина(100-400), высота(100-600),
        угол начальной и конечной точки(в градусах от 0 до 180, начало < конца);
    -затем задаётся последовательность шаров, цифрами от 0 до 1 (цвета шаров);
Уровни задаются последовательно, от первого к последнему, без разделения.
В конце нужно оставить пустую строку
